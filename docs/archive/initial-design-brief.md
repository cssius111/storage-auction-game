# 初始设计简报（历史）

> 历史材料：本文件不具有规范效力。

早期构想是一款以仓储拍卖和开箱揭晓为主题的文字策略游戏。核心关注点包括：

- 角色差异与对手表现。
- 仓库物品生成、观察、估值、竞拍、开箱和鉴定。
- 物品可见性、真假、损坏和二手价值。
- 竞拍后的节目式结算。
- 使用大模型增强角色台词与行为。

早期方案覆盖范围较广，包含大量潜在系统和内容。阶段一讨论后，项目收敛到：

- 虚构的标准化美国二手市场。
- 四个固定单人席位。
- 自动基础扫描、30 秒观察和最多 2 次重点动作。
- 三种首发仓库模板。
- 固定真相与确定性规则。
- DeepSeek 只做合法动作选择和短台词。

当前定义见 [P0 范围](../product/p0-scope.md) 和 [ADR 索引](../adr/README.md)。
