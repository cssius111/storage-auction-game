# 视频分析修订摘要（历史）

> 历史材料：本文件不具有规范效力。

早期视频分析建议用节目化结构组织玩法：角色到场、开门观察、竞拍、开箱、鉴定和记分。它还强调角色对同一线索的不同判断、竞拍姿态与揭晓反差。

阶段一保留了以下内容：

- 节目式 Episode/Session/Unit 层级。
- 快速基础扫描与有限重点观察。
- 公开竞拍行为和短台词。
- 快速揭晓与少量延迟鉴定。
- 节目利润和实际现金分开。

阶段一舍弃或推迟了以下内容：

- 对具体真实节目的模仿。
- 过多角色、组队席位和复杂关系网。
- 模型自由生成物品真相。
- 复杂商业经营、物流和实时多人玩法。
- 第一版的大规模内容与生产基础设施。

当前节奏规范见 [核心游戏循环](../product/game-loop.md)。
